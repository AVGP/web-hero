Rocket science 101

Every website is made of three things
  1. HTML defines the content and its structure
  2. CSS defines how things look
  3. Javascript gives it interactivity, it defines behaviour.

In your spaceship's cockpit, HTML would define the buttons you have available, the text for the labels and so on.

The design of those buttons - their colors, the font size, if they're transparent or hidden - would be defined by CSS.

Finally the behaviour, what the buttons do when you press them, is defined by Javascript.
