CSS - making things pretty

Now that we have built a stable home base in HTML, we want to explore the beauty of the solar system.

Until now, things have been rather boring: Black on white, standard colors and fonts, nothing fancy.

With CSS we can change that!

We have many possibilities ranging from colors, fonts and text size and the like to far more elaborate techniques like shadows, 3D transformations and animations.

A small step for a man...

You may have noticed the <style> ... </style> area in the editor.

This is the place to put your CSS rules.

CSS rules are describing the appearance of the elements.

That means, that you can make things look different - you can even decide if something is displayed at all or if it's hidden.

CSS rules are made of two things: selectors and property-value assignments. Imagine you're telling somebody to make things look a certain way.

You may say something like:

"I want all buttons to have blue text."

This pretty much translates directly into CSS:


button {
  color: blue;     
}

Try that in the editor and see the effects.

The "button" part is the selector ("all button elements"), the part in {...} contains the property-value assignments.

In the example we only set one property - the color, which is the text color of the element that match the selector - to a value (blue).
