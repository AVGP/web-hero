The orbit: HTML

The first stop on our journey through the web is the base for every further explorations:

The Hypertext markup language - or much shorter: HTML.

HTML defines the content and its meaning for the browser to make sense of all that. It defines what you see.

Examples for HTML elements are the headlines, the list on the last page, these paragraphs, links, images, buttons and so on..

It also contains non-visible parts of the structure like the body, the head (that, in the editor to the right of this) contains 

<script>
and 
<style>
areas to take Javascript and CSS later on)

Every element is wrapped in so-called tags, which are easy to spot, thanks to the angle brackets (< and >).

Tags surround their content and the ending (or closing) tag has a slash in front of its name (e.g. </h1>)

Content can be text or other tags, so nesting can be done.

As an example: You can have a headline that contains an image, or you may have a paragraph that contains a form that contains a button..

Some elements do not have content, for example a linebreak (<br />) and the slash of the closing tag is added right before the closing angle bracket of the tag.

Elements can also have attributes to describe their properties. For example links have somewhere to point to (i.e. the website they take you to, when you click them).
This is called the href (short for "hyper reference", can be any other document with an address, like another website or another HTML page of your website or an image, etc.). To define a link, you would write: 


<a href="another_document.html">Go to another document<a>

Another example of an attribute is the src attribute for images which defines the image to display like this:

<img src="some_picture.jpg" />

What else is there?

HTML knows many different tags and attributes and to give a full explanation here would take too much time and space.

There are plenty of reference guides on the different available tags, see the HTML5 for developers reference as an example.

Let's play with it!

With the help of a HTML5 reference, you should now be able to play around with the content.

Try adding a second-level headline, a button, a link or an image.

If you need some more inspiration, click the button below to see an example

<!doctype html>
<html>
  <body>
  <h1>Hello World!</h1>
  <h2>Hello visitors!</h2>
  
  <img src="http://placekitten.com/200/300" 
       width="200" height="300" 
       alt="Some image showing a cat" />

  <p>Some paragraph.</p>
  <p>And <a href="http://www.google.com">a link to Google</a>
  
  <p>
    <input id="something" />
    <button>Click me!</button>
  </p>
  </body>
</html>
