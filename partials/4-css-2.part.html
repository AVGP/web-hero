Decorate the ship

Selectors are powerful

In the last chapter we changed all buttons. But what if you want to change the background color of the self-destruct system button to red? Or what, if you want to make the text on every critical button bold, but not on the non-critical buttons?

First of all, we're gonna start with the following bit of HTML:


<!doctype html>
<html>
  <head>
      <style>
      </style>
  </head>

  <body>
    <h1>The cockpit</h1>
    <button>Self destruct</button>
    <button>Emergency stop</button>
    <button>Fire lasers</button>

    <button>Autopilot</button>
    <button>Pretend to be busy</button>
  </body>
</html>

Now the most dangerous button is obviously the self destruct button, so we want to start with that one. Let's begin with giving it an ID, like this: 

<button id="selfdestruct">Self destruct</button>

It doesn't matter what ID you use (I just used "selfdestruct", because it is obvious), as long as it's only used once on the page.

Okay, now to the CSS: We want the self destruct to have bold text and a yellow background and a dashed, red border (put it between <style>...</style>):


#selfdestruct {
  background-color: yellow;
  border: dashed red 2px;
  font-weight: bold;
}

That's pretty simple - but where do these properties and their values come from? What are the available options? The answer to that is: It's in the standard and, as with HTML, you can look these things up in a good CSS reference.

You can select elements by their ID, by using "#" and the ID you want as the selector.
